Wilt u Windows XP bij Microsoft registreren?
U kunt, terwijl u Windows on line activeert, enige persoonlijke gegevens voor de registratie van uw exemplaar van Windows opgeven.

Registratie is niet verplicht en is niet nodig voor het activeren van Windows.

Microsoft kan u, met uw toestemming en nadat u zich hebt geregistreerd, op de hoogte houden van updates voor producten, nieuwe producten, evenementen en speciale aanbiedingen.
