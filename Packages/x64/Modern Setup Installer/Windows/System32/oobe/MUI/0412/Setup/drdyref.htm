인터넷 계정을 이 컴퓨터로 복사하기 위해 인터넷 서비스 공급자를 찾는 데 도움이 필요하면 Microsoft 조회 서비스에 연결한 후, 전화로 문의할 수 있습니다.

인터넷 계정을 설정하는 데 도움이 필요하십니까?
  
  
계속하려면 아래에 정보를 입력하고 다음을 클릭하십시오. 전화를 걸지 않고 계속 진행하려면 건너뛰기를 클릭하십시오.

사용자가 거주하는 국가/지역을 선택하십시오. 국가 목록을 보려면 아래쪽 화살표를 클릭하거나 Alt+아래쪽 화살표 키를 누르십시오.(C).



지역 번호를 입력하십시오(예: 02-123-1427의 경우, 02)(A).
