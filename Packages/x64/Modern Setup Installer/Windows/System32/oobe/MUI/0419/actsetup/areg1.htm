Зарегистрироваться в корпорации Майкрософт?
При активации через Интернет Windows вы можете зарегистрировать вашу копию Windows, для этого вам потребуется сообщить некоторые сведения личного характера.

Регистрация не является обязательной и не требуется для активации Windows.

Регистрация копии Microsoft Windows позволит вам получать сообщения об обновлениях данного продукта, выходе новых продуктов, событиях и специальных предложениях.

Зарегистрировать Windows в процессе активации?

  
  

Корпорация Майкрософт принимает все меры для соблюдения конфиденциальности. Для получения дополнительной информации прочтите Соглашение о конфиденциальности регистрации Windows

Для продолжения нажмите кнопку Далее.
