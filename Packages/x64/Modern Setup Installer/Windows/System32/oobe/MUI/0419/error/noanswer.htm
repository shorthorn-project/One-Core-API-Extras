Вот несколько советов перед тем, как вы попытаетесь подключиться снова:

  • Если это правильный номер телефона, подождите несколько минут, а затем щелкните Набрать номер.

  • Если этот номер - неправильный, введите правильный номер в поле выше. Если для дозвона требуется набрать код города, введите его перед номером телефона.

  • Если это телефонный номер в другом городе (междугородний звонок), введите код выхода на междугороднюю линию (обычно, это 8) перед всеми остальными цифрами номера. Для выполнения подобного звонка необходимо, чтобы вы были подключены к междугородней линии.

Чтобы повторить попытку подключения, щелкните Набрать номер. Чтобы пропустить этот шаг, щелкните Пропустить.
