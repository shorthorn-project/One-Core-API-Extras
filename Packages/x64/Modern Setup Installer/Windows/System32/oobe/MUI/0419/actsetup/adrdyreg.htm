Активация через Интернет
Windows наберет номер телефона для активации вашей копии программного продукта в корпорации Майкрософт.

Чтобы набрать номер, укажите внизу нужные сведения и нажмите кнопку Далее. Чтобы продолжить работу без подключения к телефонной линии, нажмите кнопку Напомнить позже.

Выберите страну или регион, в котором вы сейчас находитесь. Для просмотра списка регионов щелкните кнопку со стрелкой или нажмите клавиши ALT + "СТРЕЛКА ВНИЗ".



Укажите код города (пример: 812 для (812) 555-1212).


          Чтобы начать набор номера, нажмите кнопку Далее.
