下はお住まいの地域のインターネット サービス プロバイダ (ISP) の一覧の一部です。全国的なサービスのない ISP はこの一覧に表示されません。

ISP を 1 つ選択してください(C) インターネット サービス プロバイダからアカウントの設定を取得できませんでした。[ようこそ画面] が終わったら、インターネット サービス プロバイダへ問い合わせください。

 

ご使用の ISP が一覧にないが、ユーザー名、パスワード、ISP の電話番号などのインターネット アカウント情報が既に手元にある場合は、ここをクリックしてください:
設定情報がある(H)

使う ISP が独自のソフトウェアのインストールを要求する場合は、[省略] をクリックしてください。
(ISP から既に CD を取得している場合など。)
