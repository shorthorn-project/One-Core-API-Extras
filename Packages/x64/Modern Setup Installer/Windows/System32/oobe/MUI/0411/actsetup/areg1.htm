Microsoft にユーザー登録しますか?
オンラインで Windows のライセンス認証の手続きを行っている間に、個人情報を提供して Windows のユーザー登録を行うこともできます。

ユーザー登録は省略することができ、Windows のライセンス認証の手続きには必要ありません。

ユーザー登録後、充実した製品サービス、アップデート情報やイベントなどの最新情報が Microsoft より提供されます。

Windows のライセンス認証の手続き中にユーザー登録も行いますか?

  
  

Microsoft はユーザーのプライバシーを守ります。詳細については、 Windows ユーザー登録プライバシー ポリシーをお読みください
