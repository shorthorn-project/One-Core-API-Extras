Regisztrálja a terméket a Microsoftnál?
A Windows online aktiválása során bizonyos személyes adatok megadásával regisztrálhatja is Windows-példányát.

A regisztrálás nem kötelezõ, nem feltétele a Windows aktiválásának.

A regisztrálás után, ha beleegyezik, a Microsoft tájékoztatni fogja a termékfrissítésekrõl, az új termékekrõl, az eseményekrõl és az akciós ajánlatokról.

Regisztrálni szeretné a terméket a Windows aktiválása során?

  
  

A Microsoft ügyel az Ön adatainak védelmére. Ha további információra van szüksége,  olvassa el a Windows regisztrációs adatvédelmi nyilatkozatot
