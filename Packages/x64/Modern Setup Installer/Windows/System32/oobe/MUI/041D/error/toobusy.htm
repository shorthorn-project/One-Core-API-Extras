Telefonen är upptagen eller svarar inte

Det går inte att ansluta till det här numret för att aktivera datorn.
Numret kan vara upptaget, eller så kanske det är felaktigt.

Om numret är korrekt kan du vänta en liten stund och sedan klicka på Ring igen.

Om numret är felaktigt kan du ange nya inställningar och sedan klicka på Ring igen.
    
    
    
 
 


* Ange ett alternativt nummer.
* Ange ett nummer för extern linje.
* Ange ett nummer som inaktiverar samtal väntar.
Klicka på Hoppa över om du vill fortsätta utan att aktivera datorn.
