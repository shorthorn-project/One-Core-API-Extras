Anslutningen avbröts
Det finns flera möjliga orsaker till att anslutningen har avbrutits:
  • Servern kan ha avslutat anslutningen på grund av lång tids inaktivitet. Klicka på Ring igen om du vill återansluta.
  • Någon kan ha försökt använda telefonlinjen medan du var ansluten. Kontrollera att telefonlinjen är ledig och klicka på Ring igen om du vill återansluta.
  • Om du använder samtal väntar, kan anslutningen blivit avbruten av ett inkommande samtal. Om du har numret för avstängning avsamtal väntar, skriv in det här och klickar sedan på Ring igen. Om du inte kan numret klickar du på Ring igen om du vill återansluta.

Klicka på Hoppa över om du vill fortsätta utan att utföra detta.
