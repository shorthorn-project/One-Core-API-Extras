Vill du aktivera Windows nu?

  
  
  

Om du vill vänta med att aktivera kan du fortfarande använda Windows, men du kommer att få påminnelser med jämna mellanrum.
Efter  dagar måste du aktivera Windows innan du kan fortsätta använda det.
