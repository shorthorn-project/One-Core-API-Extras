Behöver du hjälp med att hitta en Internet-leverantör?


Om du behöver hjälp med att hitta information om din Internet-leverantör för att kunna kopiera ditt Internet-konto till den här datorn, kan datorn anslutas till Microsofts referenstjänst så att Internet-leverantören kan hittas.

Behöver du hjälp med att konfigurera ditt Internet-konto?
  
  
Om du vill fortsätta anger du informationen nedan och klickar på Nästa. Om du vill fortsätta utan att ringa klickar du på Hoppa över.

Välj det land som du befinner dig i. Klicka på nedåtpilen pilen eller tryck på ALT + NEDPIL om du vill visa listan över regioner.
