Telefonen är upptagen eller svarar inte

Det nummer som ringdes upp var upptaget eller så måste du ändra numret som rings upp.

  • Om du inte är hemma kan du behöva slå ett nummer för att få en extern linje.

  • Om du inte ringer lokalt kan du behöva ändra telefonnumret.
