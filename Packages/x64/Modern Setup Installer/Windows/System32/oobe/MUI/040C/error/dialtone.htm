Pas de tonalité

La tonalité doit être détectée sur la ligne pour établir la connexion téléphonique depuis votre ordinateur. Veuillez vérifier les points suivants :
  • la ligne téléphonique est libre.
  • le fil téléphonique allant de votre ordinateur à la prise murale est bien connecté, comme indiqué dans la documentation de votre ordinateur.




    • Si vous devez rapprocher votre ordinateur de la prise téléphonique murale, éteignez-le. Lorsque votre ordinateur redémarrera, cet écran s'affichera à nouveau.

    Si vous recevez ce message une seconde fois, vous pouvez passer cette étape ; nous vous le rappellerons plus tard.

Cliquez sur Suivant pour retenter la connexion. Cliquez sur Ignorer pour passer cette étape.
