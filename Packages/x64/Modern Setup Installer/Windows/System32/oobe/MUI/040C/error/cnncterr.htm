Votre appel a été interrompu


L'appel téléphonique passé depuis votre ordinateur a été interrompu. Les causes peuvent être multiples :
  • La connexion est restée longtemps inactive.
  • Quelqu'un a utilisé le téléphone alors qu'il était en cours de connexion.
  • Si vous avez activé l'attente d'appel, un appel reçu a interrompu la connexion. Il est possible de désactiver l'attente d'appel pendant cette étape d'enregistrement, puis de la réactiver automatiquement.

    • Le code permettant de désactiver l'attente d'appel est le suivant :

Si vous recevez ce message une seconde fois, vous pouvez passer cette étape ; nous vous le rappellerons plus tard.

 

 

 

 

Cliquez sur Suivant pour retenter la connexion. Cliquez sur Ignorer pour passer cette étape.
