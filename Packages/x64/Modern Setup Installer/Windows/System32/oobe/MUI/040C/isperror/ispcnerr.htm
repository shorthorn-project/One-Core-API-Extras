La connexion a été interrompue


Plusieurs raisons peuvent expliquer pourquoi votre connexion a été interrompue. Vérifiez ce qui suit :
  • La connexion n'a pas été utilisée pendant une durée prolongée.
  • Quelqu'un a peut-être essayé d'utiliser votre ligne téléphonique alors que vous étiez connecté.
  • Si vous avez activé l'attente d'appel, il se peut que vous ayez été interrompu par un appel entrant.

    • Si vous connaissez le numéro permettant de désactiver l'attente d'appel, entrez-le dans la zone ci-dessous.

Lorsque votre ordinateur est prêt à rétablir la connexion, cliquez sur Recomposer.
Pour continuer sans configurer de service Internet sur votre ordinateur, cliquez sur Ignorer.
