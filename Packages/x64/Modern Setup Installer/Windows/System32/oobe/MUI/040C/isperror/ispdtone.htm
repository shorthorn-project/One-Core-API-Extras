La tonalité n'est pas détectée

Pour vous assurer que l'accueil de Windows compose le bon numéro, vérifiez les points suivants :
  • Le câble téléphonique entre votre ordinateur et la prise de téléphone n'est peut-être pas branché correctement. Consultez votre documentation pour vérifiez qu'il est convenablement relié.



    Remarque : si vous devez déplacer votre ordinateur pour l'approcher d'une prise téléphonique, mettez-le hors tension. Au redémarrage, l'accueil de Windows reprendra à l'endroit où il s'est arrêté.

  • La ligne de téléphone est peut-être en cours d'utilisation. Vérifiez qu'elle est disponible.

Lorsque vous êtes prêt, cliquez sur Recomposer.
Pour continuer sans paramétrer le service Internet, cliquez sur Ignorer.
