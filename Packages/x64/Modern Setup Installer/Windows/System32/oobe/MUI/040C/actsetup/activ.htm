Activation de Windows
Pour participer à la lutte contre le piratage logiciel, activez Windows maintenant. L'activation via Internet est simple et rapide. Si vous ne disposez pas d'une connexion Internet, un numéro de téléphone vous sera indiqué dans les écrans suivants. Cette copie de Windows doit être activée auprès de Microsoft avant de pouvoir continuer à l'utiliser. L'activation via Internet est simple et rapide.

L'activation est confidentielle et ne requiert aucune information personnelle.

Voulez-vous activer Windows maintenant ?

  
  
  

Si vous n'activez pas Windows maintenant, vous pourrez continuer à l'utiliser, mais vous recevrez régulièrement un message de rappel.
Après  jours, l'activation sera obligatoire pour que vous puissiez continuer à utiliser Windows.

Microsoft s'engage à respecter la confidentialité de vos données personnelles. Pour obtenir plus d'informations,  lisez la déclaration de confidentialité relative à l'activation de produit Microsoft
