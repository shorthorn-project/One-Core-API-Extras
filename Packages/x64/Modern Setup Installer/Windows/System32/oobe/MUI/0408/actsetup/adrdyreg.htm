Γίνεται προετοιμασία ηλεκτρονικής ενεργοποίησης
Τα Windows θα κάνουν τώρα κλήση σε έναν αριθμό τηλεφώνου για ενεργοποίηση των Windows στη Microsoft.

Για να προχωρήσετε, δώστε τις παρακάτω πληροφορίες και κάντε κλικ στο κουμπί Επόμενο. Για να συνεχίσετε χωρίς να πραγματοποιήσετε κλήση στην τηλεφωνική σας γραμμή, κάντε κλικ στο κουμπί Υπενθύμιση αργότερα.

Επιλέξτε τη χώρα/περιοχή στην οποία βρίσκεστε τώρα. Κάντε κλικ στο κάτω βέλος ή πιέστε τα πλήκτρα ALT + "ΚΑΤΩ ΒΕΛΟΣ" για να δείτε τη λίστα των περιοχών.



Πληκτρολογήστε τον υπεραστικό κωδικό του τηλεφώνου σας. (π.χ., 425 στο 425-555-1212).


          Για να ξεκινήσετε την κλήση κάντε κλικ στο Επόμενο.
