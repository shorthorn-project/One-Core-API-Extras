Γίνεται κλήση:

Μερικές συμβουλές πριν προσπαθήσετε να συνδεθείτε πάλι:

  • Αν αυτός είναι ο σωστός αριθμός τηλεφώνου, περιμένετε μερικά λεπτά και κάντε κλικ στο κουμπί Επανάληψη κλήσης.

  • Αν αυτός ο αριθμός τηλεφώνου δεν είναι ο σωστός, πληκτρολογήστε τον σωστό αριθμό στο παραπάνω πλαίσιο. Αν χρειάζεται να καλέσετε έναν επιπλέον αριθμό για πρόσβαση σε εξωτερική γραμμή, συμπεριλάβετέ τον στην αρχή του αριθμού τηλεφώνου.

  • Αν αυτός ο αριθμός τηλεφώνου είναι εκτός της τοπικής σας περιοχής, συμπεριλάβετε 1 + τον κωδικό περιοχής. Χρειάζεστε υπηρεσία υπεραστικών κλήσεων για να ολοκληρώσετε την κλήση.

Για να προσπαθήσετε πάλι να συνδεθείτε, κάντε κλικ στο κουμπί Επανάληψη κλήσης. Για να παραλείψετε αυτό το βήμα, κάντε κλικ στο κουμπί Παράλειψη.
