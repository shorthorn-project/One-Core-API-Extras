Προσωρινά η υπηρεσία παροχής Internet δεν είναι διαθέσιμη

Οι υπηρεσίες νέου λογαριασμού, για την υπηρεσία παροχής Internet (ISP) που επιλέξατε, δεν είναι αυτήν τη στιγμή διαθέσιμες. Για να προσπαθήσετε ξανά, περιμένετε λίγα λεπτά και κάντε κλικ στο κουμπί Επανάληψη κλήσης.

Για να επιλέξετε άλλη υπηρεσία παροχής Internet, κάντε κλικ στο κουμπί Προηγούμενο.
Για να συνεχίσετε χωρίς να ορίσετε την υπηρεσία Internet, κάντε κλικ στο κουμπί Παράλειψη.
