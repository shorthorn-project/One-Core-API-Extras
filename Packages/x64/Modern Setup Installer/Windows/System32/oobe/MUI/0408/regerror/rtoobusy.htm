Ο αριθμός τηλεφώνου είναι απασχολημένος ή δεν απάντησε

Ο αριθμός που καλέσατε ήταν απασχολημένος ή ίσως χρειάζεται να τροποποιήσετε τον αριθμό αυτό.

  • Εάν βρίσκεστε εκτός σπιτιού, ίσως χρειαστεί να καλέσετε έναν αριθμό για εξωτερική γραμμή.

  • Εάν πραγματοποιείτε κλήση εκτός της περιοχής σας, ίσως χρειαστεί να τροποποιήσετε τον αριθμό τηλεφώνου.

Γίνεται κλήση:

  • Εάν ο προηγούμενος αριθμός είναι σωστός, περιμένετε μερικά λεπτά και κάντε κλικ στο κουμπί Επανάληψη κλήσης.

  • Εάν ο αριθμός δεν είναι σωστός, τροποποιήστε τη ρύθμιση παραπάνω και κάντε κλικ στο κουμπί Επανάληψη κλήσης.

  • Για να επαναφέρετε τον αρχικό αριθμό τηλεφώνου που καλέσατε, κάντε κλικ στο κουμπί Επαναφορά.

Για να συνεχίσετε χωρίς να εκτελέσετε αυτό το βήμα, κάντε κλικ στην επιλογή Παράλειψη.
