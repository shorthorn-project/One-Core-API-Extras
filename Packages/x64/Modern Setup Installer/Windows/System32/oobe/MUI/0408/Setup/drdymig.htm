Γίνεται κλήση στην υπηρεσία σας παροχής Internet...

Τώρα θα κάνουμε ένα τηλεφώνημα για να ρυθμίσουμε τον υπάρχοντα λογαριασμό σας Internet σε αυτόν τον υπολογιστή.   Κάντε κλικ στο κουμπί Επόμενο για ξεκινήσετε την κλήση.



Για να προχωρήσετε, δώστε τις παρακάτω πληροφορίες και κάντε κλικ στο κουμπί Επόμενο. Για να συνεχίσετε χωρίς να πραγματοποιήσετε κλήση στην τηλεφωνική σας γραμμή, κάντε κλικ στο κουμπί Παράλειψη.

Επιλέξτε τη χώρα/περιοχή στην οποία βρίσκεστε τώρα. Κάντε κλικ στο κάτω βέλος ή πιέστε τα πλήκτρα ALT + "ΚΑΤΩ ΒΕΛΟΣ" για να δείτε τη λίστα των περιοχών.

Πληκτρολογήστε τον υπεραστικό κωδικό του τηλεφώνου σας. (π.χ., 031 στο 031-654321).
