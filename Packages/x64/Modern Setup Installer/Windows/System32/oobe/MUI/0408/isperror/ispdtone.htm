Δεν εντοπίστηκε ήχος τηλεφωνικής επιλογής

Για να εξασφαλίσετε ότι από την Υποδοχή των Windows μπορεί να πραγματοποιηθεί σωστά μια κλήση, ελέγξτε τα ακόλουθα:
  • Το καλώδιο του τηλεφώνου, μεταξύ του υπολογιστή σας και της τηλεφωνικής πρίζας, ίσως δεν είναι συνδεδεμένο σωστά. Συμβουλευθείτε το εγχειρίδιό σας, για να εξασφαλίσετε ότι το καλώδιο έχει συνδεθεί σωστά.



    Σημείωση: Εάν πρέπει να μετακινήσετε τον υπολογιστή σας, ώστε να βρίσκεται κοντά σε τηλεφωνική πρίζα, κλείστε την τροφοδοσία του υπολογιστή. Όταν ξεκινήσετε πάλι τον υπολογιστή σας, η Υποδοχή των Windows θα ξεκινήσει από εκεί που σταματήσατε.

  • Ενδέχεται να χρησιμοποιείται η τηλεφωνική σας γραμμή. Βεβαιωθείτε ότι η τηλεφωνική γραμμή είναι διαθέσιμη.

Όταν είστε έτοιμοι να συνεχίσετε, κάντε κλικ στο κουμπί Επανάληψη κλήσης.
Για να συνεχίσετε χωρίς να εγκαταστήσετε υπηρεσία Internet, κάντε κλικ στο κουμπί Παράλειψη.
