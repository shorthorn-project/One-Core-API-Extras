Σας ευχαριστούμε

Συγχαρητήρια, είστε έτοιμοι! Να τι καταφέρατε:
Για να μάθετε για τα συναρπαστικά νέα χαρακτηριστικά των Windows XP, κάντε μια περιήγηση στο προϊόν. Μπορείτε επίσης να βρείτε χρήσιμες πληροφορίες στο Κέντρο Βοήθειας και Υποστήριξης. Αυτές οι επιλογές υπάρχουν στο μενού Έναρξη.
Η Υποδοχή των Windows πρέπει να κάνει επανεκκίνηση του υπολογιστή σας για να οριστικοποιήσει τις ρυθμίσεις σας.
Κάντε κλικ στο κουμπί Τέλος να επανεκκινήσετε τον υπολογιστή σας και να ξεκινήσετε να εργάζεστε με τα Windows.
