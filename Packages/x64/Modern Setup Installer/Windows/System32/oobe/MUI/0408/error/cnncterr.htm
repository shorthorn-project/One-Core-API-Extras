Η κλήση σας διακόπηκε


Η τηλεφωνική κλήση από τον υπολογιστή σας διακόπηκε πριν ολοκληρωθεί. Μερικοί πιθανοί λόγοι είναι:
  • Η σύνδεση δεν έχει χρησιμοποιηθεί για πολύ καιρό.
  • Κάποιος προσπάθησε να χρησιμοποιήσει το τηλέφωνό σας καθώς γινόταν σύνδεση.
  • Αν έχετε αναμονή κλήσης, μια εισερχόμενη κλήση διέκοψε τη σύνδεση. Η αναμονή κλήσης μπορεί να απενεργοποιηθεί αυτόματα μέχρι να ολοκληρωθεί αυτό το βήμα της δήλωσης και στη συνέχεια να ενεργοποιηθεί πάλι.

    • Ο κωδικός για απενεργοποίηση της αναμονής κλήσης είναι:

Αν λάβετε αυτό το μήνυμα δεύτερη φορά, μπορείτε να παραλείψετε αυτό το βήμα και θα σας γίνει υπενθύμιση αργότερα.

 

 

 

 

Για να προσπαθήσετε πάλι να συνδεθείτε κάντε κλικ στο κουμπί "Επόμενο". Για να παραλείψετε αυτό το βήμα, κάντε κλικ στο κουμπί "Παράλειψη".
