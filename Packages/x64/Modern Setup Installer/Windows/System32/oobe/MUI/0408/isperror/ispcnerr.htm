Η σύνδεση διακόπηκε


Υπάρχουν διάφοροι λόγοι που είναι υπεύθυνοι για τη διακοπή της σύνδεσής σας. Ελέγξτε τα ακόλουθα:
  • Η σύνδεση δεν έχει χρησιμοποιηθεί για παρατεταμένο χρονικό διάστημα.
  • Ενδέχεται κάποιος να έχει προσπαθήσει να χρησιμοποιήσει την τηλεφωνική σας γραμμή ενώ ήσασταν συνδεδεμένοι.
  • Εάν έχετε αναμονή κλήσης, ενδέχεται να έχετε διακοπεί από μια εισερχόμενη κλήση.

    • Αν γνωρίζετε τον αριθμό για απενεργοποίηση της αναμονής κλήσης, πληκτρολογήστε τον στο παρακάτω πλαίσιο.

Όταν ο υπολογιστής σας είναι έτοιμος να επανασυνδεθεί, κάντε κλικ στο κουμπί Επανάληψη κλήσης.
Για να συνεχίσετε, χωρίς να ρυθμίσετε τις παραμέτρους της υπηρεσίας παροχής Internet στον υπολογιστή σας, κάντε κλικ στο κουμπί Παράλειψη.
