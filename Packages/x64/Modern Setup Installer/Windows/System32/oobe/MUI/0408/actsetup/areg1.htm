Θέλετε να κάνετε δήλωση στη Microsoft;
Κατά την ηλεκτρονική ενεργοποίηση των Windows μπορείτε να καταχωρήσετε κάποιες προσωπικές σας πληροφορίες, για να δηλώσετε το αντίγραφο των Windows.

Η δήλωση είναι προαιρετική και δεν απαιτείται για την ενεργοποίηση των Windows.

Αφού κάνετε δήλωση, η Microsoft μπορεί, με την άδειά σας, να σας ειδοποιεί για ενημερωμένες εκδόσεις προϊόντων, νέα προϊόντα, συμβάντα και ειδικές προσφορές.

Θέλετε να κάνετε δήλωση ενώ ενεργοποιείτε τα Windows;

  
  

Η Microsoft έχει δεσμευτεί για το προσωπικό απόρρητο. Για περισσότερες πληροφορίες,  διαβάστε τη δήλωση απορρήτου για τη δήλωση των Windows

Για να συνεχίσετε, κάντε κλικ στο κουμπί "Επόμενο".
