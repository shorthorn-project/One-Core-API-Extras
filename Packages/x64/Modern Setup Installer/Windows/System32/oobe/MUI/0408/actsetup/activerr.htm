Ενεργοποίηση των Windows
Η ηλεκτρονική ενεργοποίηση των Windows δεν ήταν δυνατή. Κάντε κλικ στο κουμπί Τηλέφωνο παρακάτω, για οδηγίες σχετικά με τη χρήση του τηλεφώνου σας για την ενεργοποίηση των Windows μέσω μιας υπηρεσίας εξυπηρέτησης πελατών. Πατήστε το κουμπί Επανάληψη, για να γίνει επανάληψη της προσπάθειας.
