La oss aktivere Windows
Hvis du vil være med å redusere piratkopiering av programvare, aktiverer du Windows-eksemplaret nå. Aktivering over Internett er raskt og enkelt. Dette eksemplaret av Windows må aktiveres hos Microsoft før du kan fortsette å bruke det. Aktivering over Internett er raskt og enkelt.

Du trenger ikke oppgi navn eller annen personlig informasjon når du aktiverer Windows.

Vil du aktivere Windows nå?

  
  
  

Hvis du venter med å aktivere, kan du fortsatt bruke Windows, men du mottar jevnlige påminnelser.
Etter  dager må du aktivere Windows for at du skal kunne fortsette å bruke det.

Microsoft går inn for å ivareta den enkeltes personvern. Hvis du vil ha mer informasjon, se  les Personvernerklæring for produktaktivering for Windows

Klikk Neste for å fortsette.
