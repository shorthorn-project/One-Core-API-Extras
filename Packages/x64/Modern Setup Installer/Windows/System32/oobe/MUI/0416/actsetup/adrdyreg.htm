Preparando-se para a ativação on-line
Agora o Windows vai discar um número de telefone para ativar o Windows na Microsoft.

Para continuar, forneça as informações abaixo e clique em Avançar. Para continuar sem fazer a discagem telefônica, clique em Lembrar mais tarde.

Selecione o país ou região onde você está agora. Clique na seta para baixo ou pressione as teclas ALT + "seta para baixo" para exibir a lista de regiões.



Digite o código de área do seu telefone (por exemplo, 99 em 99-222-3333).


          Para começar a discagem, clique em Avançar.
