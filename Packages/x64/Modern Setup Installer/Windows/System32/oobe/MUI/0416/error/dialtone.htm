Não há sinal de linha

É necessário ter um sinal de linha para se estabelecer a conexão telefônica do seu computador. Certifique-se de que foram atendidas as seguintes condições:
  • A linha telefônica está disponível.
  • O fio telefônico entre o computador e a tomada do telefone está conectado corretamente, conforme indicado na documentação do computador.




    • Se você precisar mover o computador para perto de uma tomada telefônica, desligue-o primeiro. Quando o computador for reiniciado, o Windows continuará do ponto em que você parou.

    Se você receber esta mensagem uma segunda vez, poderá ignorar esta etapa e o lembraremos mais tarde.

Para tentar se conectar novamente, clique em 'Rediscar'. Para ignorar esta etapa, clique em 'Ignorar'.
