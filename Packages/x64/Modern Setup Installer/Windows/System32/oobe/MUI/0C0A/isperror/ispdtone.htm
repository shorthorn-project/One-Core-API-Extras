No se detectó el tono de marcado

Para asegurarse de que la bienvenida de Windows puede marcar correctamente, compruebe:
  • Es posible que el cable telefónico entre el enchufe del teléfono y su PC no esté correctamente conectado. Consulte la documentación para asegurarse de que está correctamente conectado.



    Nota: si necesita mover el equipo para conectarlo a la red telefónica, apáguelo. Al reiniciar su equipo, la bienvenida de Windows continuará donde se quedó.

  • Es posible que la línea telefónica esté en uso. Compruebe que la línea esté libre.

Haga clic en Volver a marcar cuando esté listo para continuar.
Para continuar sin instalar el servicio de Internet, haga clic en Omitir.
