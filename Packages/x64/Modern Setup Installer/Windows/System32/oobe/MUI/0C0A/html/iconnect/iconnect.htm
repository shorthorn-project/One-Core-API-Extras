Configurar su cuenta de Internet

Necesitará conseguir la información siguiente de su proveedor de servicios de Internet (ISP) para rellenar esta página.

     Su nombre de usuario: (Ejemplo: anadelvalle)
     Su contraseña de usuario: (Ejemplo: 123xyz)
     El número de teléfono de su Proveedor de servicios de Internet:    (Ejemplo: 800 | 555-1234)


Si la información siguiente le ha sido proporcionada, escríbala más abajo. Esta información es opcional; si no la tiene, omita esta sección.

    
Obtener automáticamente una dirección IP (DHCP):
   * Escriba una dirección IP válida

     Dirección de protocolo Internet (IP) estática:

  Intervalo: 1-223.0-255.0-255.0-255

    
Obtener automáticamente DNS (DHCP):
   * Escriba una dirección DNS válida

     Servidor de nombres de dominio (DNS) preferido:   

  Intervalo: 1-223.0-255.0-255.0-255
     Servidor de nombres de dominio (DNS) alternativo:   (Opcional)
