¿Desea ayuda para buscar un proveedor de servicios Internet?


Si necesita ayuda para encontrar información acerca de su proveedor de servicios Internet (ISP) y así poder copiar su cuenta de Internet en este equipo, Windows puede efectuar una llamada telefónica al Servicio de referencia de Microsoft para buscar su ISP.

¿Necesita ayuda para configurar su cuenta de Internet?
  
  
Para continuar, proporcione la siguiente información y haga clic en Siguiente. Para continuar sin llamar por teléfono, haga clic en Omitir.

Seleccione el país o región donde se encuentre ahora. Haga clic en la flecha abajo o presione las teclas ALT + FLECHA ABAJO para ver la lista de regiones.



Escriba aquí el código telefónico de área (por ejemplo, 425 en 425-555-1212).
