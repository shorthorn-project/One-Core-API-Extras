Selezionare il proprio provider di servizi Internet Rivolgersi al provider di servizi Internet
Quello che segue è un elenco parziale di provider di servizi Internet (ISP, Internet Service Provider), limitato a fornitori che operano a livello nazionale.

Scegliere un ISP (facendo clic sul nome). Impossibile recuperare le impostazioni dell'account dal provider di servizi Internet. Una volta completato il processo, contattare il provider di servizi Internet per assistenza nella configurazione dell'account Internet sul computer.

 

Se già si è in possesso delle informazioni dell'account Internet, compresi nome utente, password, e numero di telefono dell'ISP, fare clic qui:
Impostazioni

Per continuare l'impostazione di un account Internet con l'ISP selezionato, fare clic su Avanti.

Se l'ISP che si desidera utilizzare richiede di installare il proprio software, fare clic su Ignora.
(È questo il caso se l'ISP ha fornito un CD).
Configurare quindi il computer dopo aver completato la configurazione di Windows.
