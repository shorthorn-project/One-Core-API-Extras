מעוניין להירשם ב- Microsoft?
תוך כדי הפעלת Windows באופן מקוון, באפשרותך גם לספק מידע אישי כדי לרשום את עותק Windows שברשותך.

הרישום הוא אופציונלי ואינו נדרש לשם הפעלת Windows.

לאחר שתבצע את הרישום, תהיה ל- Windows אפשרות, בהסכמתך, לדווח לך על עדכונים למוצרים, מוצרים חדשים, אירועים והצעות מיוחדות.

האם ברצונך להירשם בזמן ההפעלה של Windows?

  
  

Microsoft מתחייבת לשמור על פרטיותך. לקבלת מידע נוסף,  קרא את הצהרת הפרטיות של רישום Windows

כדי להמשיך, לחץ על הבא.
