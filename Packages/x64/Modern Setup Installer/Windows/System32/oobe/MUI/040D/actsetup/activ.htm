‏‏בוא נפעיל את Windows
‏‏כדי לסייע בהקטנת התופעה של תוכנות פיראטיות, נא בצע הפעלה של עותק Windows שברשותך כעת. הפעלה באינטרנט היא קלה ומהירה. ‏‏יש לבצע הפעלה של עותק זה של Windows עם Microsoft לפני שניתן להמשיך להשתמש בו. הפעלה באינטרנט היא קלה ומהירה.

אינך חייב לספק את שמך או מידע אישי אחר בעת הפעלת Windows.

‏‏האם ברצונך להפעיל כעת את Windows?

  
  
  

‏‏אם תמתין ולא תפעיל, עדיין תוכל להשתמש ב- Windows, אך תקבל תזכורות תקופתיות.
לאחר  ימים, יהיה עליך להפעיל את Windows לפני שתוכל להמשיך להשתמש בו.

Microsoft מחויבת לשמירה על פרטיותך. לקבלת מידע נוסף,  קרא את הצהרת הפרטיות להפעלת מוצר Windows

כדי להמשיך, לחץ על הבא.
