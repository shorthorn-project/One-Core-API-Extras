設定您的網際網路帳戶

您必須向您的網際網路服務提供者 (ISP) 索取下列資訊，才能填寫這個畫面。

     您的使用者名稱(U): (範例: johndoe)
     您的使用者密碼(W): (範例: 123xyz)
     您的 ISP 電話號碼(P):    (範例: 800 | 555-1234)


如果您有下列資訊，請在下面方塊中輸入。這些資訊可省略，所以如果您沒有這些資訊，可以跳過這個部分。

    
自動取得 IP (DHCP)(I):
   * 請輸入正確的 IP 位址

     靜態 IP 位址(C):

  範圍: 1-223.0-255.0-255.0-255

    
自動取得 DNS (DHCP)(D):
   * 請輸入正確的 DNS 位址

     慣用的網域名稱伺服器 (DNS)(R):   

  範圍: 1-223.0-255.0-255.0-255
     其他的網域名稱伺服器 (DNS)(A):   (可省略)
