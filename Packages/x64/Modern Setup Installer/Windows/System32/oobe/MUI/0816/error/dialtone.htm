Não há sinal de linha

Necessitamos de um sinal de linha para poder estabelecer a ligação telefónica do computador. Certifique-se da existência destas condições:
  • A linha telefónica está disponível para ser utilizada.
  • O cabo do telefone entre o computador e a tomada do telefone estão ligados correctamente, de acordo com a documentação do computador.




    • Se tiver de mover o computador para mais perto de uma tomada de telefone, primeiro desligue o computador. Quando reiniciar o computador, o procedimento será retomado onde foi interrompido.

    Se receber esta mensagem em duplicado, pode ignorar este passo e lembrá-lo-emos mais tarde.

Para tentar ligar mais tarde, clique em 'Marcar novamente'. Para ignorar este passo, clique em 'Ignorar'.
