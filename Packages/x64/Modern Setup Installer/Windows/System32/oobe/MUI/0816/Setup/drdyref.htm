Precisa de ajuda para localizar um fornecedor de serviços Internet?


Se necessitar de ajuda para localizar informações sobre o seu fornecedor de serviços Internet (ISP) para copiar a sua conta da Internet para este computador, podemos fazer uma chamada para o Microsoft Referral Service para localizar o seu ISP.

Precisa de ajuda para configurar a sua conta da Internet?
  
  
Para avançar, forneça as informações abaixo e clique em Seguinte. Para continuar sem marcar a linha telefónica, clique em Ignorar.

Seleccione o pais/região onde se encontra agora. Clique na seta para baixo ou prima as teclas ALT + "SETA PARA BAIXO" para ver a lista de regiões.



Introduza o indicativo. (por exemplo, 21 em 212400977).
