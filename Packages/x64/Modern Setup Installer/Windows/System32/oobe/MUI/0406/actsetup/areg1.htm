Registrer hos Microsoft?
Mens du aktiverer Windows online, kan du også angive nogle personlige oplysninger for at registrere din kopi af Windows.

Registrering er valgfrit og er ikke nødvendig for at aktivere Windows.

Når du har registreret, kan Microsoft give dig besked om produktopdateringer, nye produkter, kampagner og specielle tilbud, hvis du tillader det.

Vil du også registrere, mens du aktiverer Windows?

  
  

Microsoft beskytter dine personlige oplysninger. Du kan få yderligere oplysninger ved at  læse erklæringen om beskyttelse af personlige oplysninger ved registrering af Windows
