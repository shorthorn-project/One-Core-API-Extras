Êtes-vous prêt à ouvrir votre compte Internet ?

Nous allons maintenant appeler pour vous ouvrir un compte Internet.


Pour continuer, indiquez les informations ci-dessous et cliquez sur Suivant. Pour poursuivre sans utiliser votre ligne téléphonique pour composer le numéro, cliquez sur Ignorer.

Sélectionnez le pays/la région où vous vous trouvez actuellement. Cliquez sur la flèche pointant vers le bas ou appuyez sur les touches ALT + "BAS" pour afficher la liste des régions.

Entrez votre indicatif urbain (par exemple, 01 dans 01 42 23 57 89).
