Paramétrez votre compte Internet

Vous devrez obtenir les informations suivantes de votre fournisseur de services Internet (ISP) afin de remplir cette page.

     Votre nom d'utilisateur : (Exemple : jeanmartin)
     Votre mot de passe d'utilisateur : (Exemple : 123xyz)
     Le numéro de téléphone de votre fournisseur d'accès Internet :    (Exemple : 800 | 555-1234)


Si les informations suivantes vous ont été fournies, entrez-les ci-dessous. Ces informations sont optionnelles, si vous ne les avez pas, ignorez cette section.

    
Obtenir une adresse IP automatiquement (DHCP) :
   * Entrez une adresse IP valide

     Adresse IP statique :

  Plage : 1-223.0-255.0-255.0-255

    
Obtenir un serveur DNS automatiquement (DHCP) :
   * Entrez une adresse DNS valide

     Serveur DNS préféré :   

  Plage : 1-223.0-255.0-255.0-255
     Serrveur de noms de domaines secondaire :   (facultatif)
