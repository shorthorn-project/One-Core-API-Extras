Sélectionnez votre fournisseur de services Internet. Contactez votre fournisseur de services Internet
La liste ci-dessous contient des fournisseurs de services Internet situés dans votre région. Les fournisseurs de services Internet n'ayant pas de couverture nationale n'apparaissent pas dans cette liste.

Choisissez un fournisseurs de services Internet en cliquant sur son nom. L'accueil de Windows ne peut pas récupérer vos paramètres de compte à partir de votre fournisseur de services Internet. À la fin de l'accueil Windows, contactez votre fournisseur de services Internet pour qu'il vous aide à configurer votre compte Internet sur cet ordinateur.

 

Si votre ISP n'est pas dans la liste mais que vous avez déjà les informations concernant votre compte Internet avec vous, en incluant votre nom d'utilisateur, votre mot de passe, et le numéro de téléphone de votre fournisseur de services Internet, cliquez ici :
Fournir mes paramètres

Pour poursuivre le paramétrage d'un compte Internet avec le fournisseur de services que vous avez sélectionné, cliquez sur Suivant.

Si le fournisseur que vous voulez utiliser nécessite que vous utilisiez son propre logiciel, cliquez sur Ignorer.
(Vous saurez si c'est le cas si vous avez déjà un CD de ce fournisseur.)
Puis paramétrez votre ordinateur pour avoir accès à Internet quand vous avez fini de paramétrer Windows.
