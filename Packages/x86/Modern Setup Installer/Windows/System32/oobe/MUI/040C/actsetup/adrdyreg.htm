Préparation de l'activation en ligne
Windows va maintenant composer un numéro de téléphone afin d'activer Windows auprès de Microsoft.

Pour continuer, indiquez les informations ci-dessous et cliquez sur Suivant. Pour poursuivre sans composer le numéro sur votre téléphone, cliquez sur Plus tard.

Sélectionnez le pays ou la région où vous vous trouvez. Cliquez sur la flèche vers le bas, ou appuyez sur ALT + "BAS" pour afficher la liste des régions.



Entrez votre indicatif urbain (par exemple, 01 dans 01 42 23 58 98).


          Pour commencer la numérotation, cliquez sur Suivant.
