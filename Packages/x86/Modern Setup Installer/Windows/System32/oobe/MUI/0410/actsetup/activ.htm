Attivazione di Windows
Per contribuire a combattere la pirateria informatica, si prega di attivare subito questa copia di Windows. L'attivazione via Internet è semplice e veloce. È necessario attivare questa copia di Windows presso MIcrosoft per poter continuare a utilizzarla. L'attivazione via Internet è semplice e veloce.

Non è necessario fornire nome o altre informazioni personali all'atto dell'attivazione di Windows.

Attivare subito Windows?

  
  
  

Se si rimanda l'attivazione, sarà ancora possibile utilizzare Windows, ma si riceveranno periodicamente dei promemoria.
Dopo  giorni, sarà necessario attivare Windows per poter continuare ad usarlo.

Microsoft garantisce la tutela dei dati personali. Per ulteriori informazioni,  leggere l'informativa sul trattamento dei dati personali per l'Attivazione di un prodotto Windows
