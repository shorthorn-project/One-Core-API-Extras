La chiamata è stata interrotta


La telefonata dal computer è stata interrotta prima di essere completata. Alcune delle cause possibili sono:
  • La connessione non è stata utilizzata per un lungo periodo di tempo.
  • Altre persone potrebbero avere tentato di utilizzare il telefono durante la connessione.
  • Se si utilizza l'avviso di chiamata, la connessione potrebbe essere stata interrotta da una chiamata in arrivo. È possibile disattivare automaticamente l'avviso di chiamata durante questa fase della registrazione, riattivandolo a fase conclusa.

Se il messaggio compare una seconda volta, è possibile ignorare il passaggio.
