Configurazione impostazioni di rete
Se per la connessione a Internet è necessario configurare un server proxy, immettere le impostazioni di seguito. Contattare l'amministratore di rete qualora non si conoscano le impostazioni da utilizzare.

Configurazione automatica
La configurazione automatica potrebbe aver sovrascritto quella manuale. Qualora vengano immesse le impostazioni del proxy, deselezionare la casella di controllo etichettata "Rileva automaticamente impostazioni" per fare in modo che le modifiche siano applicate.
