‏‏מתכונן להפעלה מקוונת
‏‏Windows יחייג כעת מספר טלפון כדי להפעיל את Windows עם Microsoft.

כדי להמשיך, הזן את המידע להלן ולאחר מכן לחץ על הבא. כדי להמשיך מבלי לחייג מקו הטלפון שלך, לחץ על הזכר לי מאוחר יותר.

בחר במדינה/אזור בהם אתה נמצא כעת. לחץ על החץ למטה או הקש ‎ ALT + "חץ למטה"‎ כדי להציג את רשימת האזורים.



הזן את אזור החיוג של מספר הטלפון שלך (למשל, 425 במספר ‎425-555-1212).


          כדי להתחיל בחיוג, לחץ על הבא.
