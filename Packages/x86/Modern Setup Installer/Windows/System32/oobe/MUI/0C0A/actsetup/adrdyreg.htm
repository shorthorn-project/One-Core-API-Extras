Preparándose para activar en línea
Windows marcará un número de teléfono para activar Windows a través de Microsoft.

Para continuar, proporcione la siguiente información y haga clic en Siguiente. Para continuar sin llamar por teléfono, haga clic en Recordar después.

Seleccione el país o la región donde se encuentre ahora. Haga clic en la flecha abajo o presione las teclas ALT + "FLECHA ABAJO" para ver la lista de regiones.



Escriba aquí el código telefónico de área (por ejemplo, 425 en 425-555-1212).


          Haga clic en Siguiente para empezar a marcar.
