Η σύνδεση διακόπηκε
Υπάρχουν διάφοροι πιθανοί λόγοι που είναι υπεύθυνοι για τη διακοπή της σύνδεσης:
  • Ο διακομιστής μπορεί να διέκοψε τη σύνδεση λόγω παρατεταμένης περιόδου αδράνειας. Για να συνδεθείτε πάλι, κάντε κλικ στο κουμπί Επανάληψη κλήσης.
  • Ενδέχεται κάποιος να έχει προσπαθήσει να χρησιμοποιήσει την τηλεφωνική σας γραμμή, ενώ ήσασταν συνδεδεμένοι. Για να συνδεθείτε πάλι, βεβαιωθείτε ότι η τηλεφωνική γραμμή είναι διαθέσιμη και κάντε κλικ στο κουμπί Επανάληψη κλήσης.
  • Αν έχετε αναμονή κλήσης, η σύνδεση ίσως διακόπηκε από μια εισερχόμενη κλήση. Αν γνωρίζετε τον αριθμό για απενεργοποίηση της αναμονής κλήσης, πληκτρολογήστε τον εδώ και κατόπιν κάντε κλικ στο κουμπί Επανάληψη κλήσης. Εάν δεν γνωρίζετε τον αριθμό, κάντε κλικ στο κουμπί Επανάληψη κλήσης, για να συνδεθείτε πάλι.

Για να συνεχίσετε χωρίς να εκτελέσετε αυτό το βήμα, κάντε κλικ στην επιλογή Παράλειψη.
