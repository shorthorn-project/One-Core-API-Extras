Δεν υπάρχει τόνος κλήσης

Χρειάζεται ένας τόνος κλήσης για τη δημιουργία της τηλεφωνικής σύνδεσης από τον υπολογιστή σας. Βεβαιωθείτε ότι ισχύουν τα παρακάτω:
  • Η τηλεφωνική γραμμή είναι διαθέσιμη για χρήση.
  • Το τηλεφωνικό καλώδιο μεταξύ του υπολογιστή σας και της τηλεφωνικής πρίζας είναι σωστά συνδεδεμένο, όπως περιγράφεται στην τεκμηρίωση του υπολογιστή σας.




    • Εάν πρέπει να μετακινήσετε τον υπολογιστή σας πιο κοντά σε μια τηλεφωνική πρίζα, κλείστε πρώτα την τροφοδοσία ρεύματος. Όταν επανεκκινήσετε τον υπολογιστή σας, θα συνεχίσουμε από το σημείο που σταματήσαμε.

    Αν λάβετε αυτό το μήνυμα δεύτερη φορά, μπορείτε να παραλείψετε αυτό το βήμα και θα σας γίνει υπενθύμιση αργότερα.

Για να προσπαθήσετε πάλι να συνδεθείτε κάντε κλικ στο κουμπί Επανάληψη Κλήσης. Για να παραλείψετε αυτό το βήμα, κάντε κλικ στο κουμπί Παράλειψη.
