Προσωρινά, δεν είναι δυνατή η ολοκλήρωση της ηλεκτρονικής δήλωσης

Οι υπηρεσίες δήλωσης δεν είναι προσωρινά διαθέσιμες. Για να ολοκληρώσετε τη διαδικασία δήλωσης, περιμένετε μερικά λεπτά και κάντε κλικ στο κουμπί Επανάληψη κλήσης. Εάν λάβετε αυτό το σφάλμα για δεύτερη φορά, επικοινωνήστε με τον κατασκευαστή του υπολογιστή σας.
