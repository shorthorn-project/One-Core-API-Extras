Ο αριθμός τηλεφώνου είναι απασχολημένος ή δεν απάντά

Το πρόγραμμα εγκατάστασης δεν μπορεί να συνδεθεί με αυτόν τον αριθμό για να ενεργοποιήσει τον υπολογιστή σας.
Ίσως το τηλέφωνο είναι απασχολημένο ή πιθανόν ο αριθμός είναι εσφαλμένος.

Εάν ο αριθμός είναι σωστός, περιμένετε για μερικά λεπτά και κάντε κλικ στο Επανάληψη Κλήσης.

Αν ο αριθμός είναι εσφαλμένος, εισαγάγετε τις νέες ρυθμίσεις και κάντε κλικ στο κουμπί Επανάληψη κλήσης.
    
    
    
 
 


* Εισαγάγετε έναν εναλλακτικό αριθμό.
* Εισαγάγετε έναν αριθμό για πρόσβαση σε εξωτερική γραμμή.
* Εισαγάγετε έναν αριθμό για απενεργοποίηση αναμονής κλήσης.
Για να συνεχίσετε χωρίς να ενεργοποιήσετε τον υπολογιστή σας, κάντε κλικ στο κουμπί Παράλειψη.
