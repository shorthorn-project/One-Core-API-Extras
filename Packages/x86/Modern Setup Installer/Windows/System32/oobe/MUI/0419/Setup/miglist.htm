Выберите поставщика услуг Интернета Обратитесь к поставщику услуг Интернета
Ниже указан частичный список общенациональных поставщиков услуг Интернета (ISP) в вашем городе или области. Поставщики, которые не оказывают общенациональных услуг в списке отсутствуют.

Выберите поставщика услуг Интернета, щелкнув его имя. Программе приветствия Windows не удается загрузить параметры вашей учетной записи у поставщика услуг Интернета. После завершения программы свяжитесь с поставщиком услуг Интернета, чтобы получить помощь в настройке учетной записи на данном компьютере.

 

Если ваш поставщик услуг Интернета не находится в списке ISP, но у вас есть вся необходимая информация (имя учетной записи, пароль и номер телефона поставщика Интернета), щелкните здесь:
Параметры подключения известны

Чтобы продолжить установку параметров учетной записи Интернета у выбранного поставщика услуг, щелкните Далеe.

Если используемый поставщик требует установить его программное обеспечение (например, с распространяемого им компакт-диска), щелкните Пропустить.
Установите доступ к Интернету после окончания настройки параметров Windows.
