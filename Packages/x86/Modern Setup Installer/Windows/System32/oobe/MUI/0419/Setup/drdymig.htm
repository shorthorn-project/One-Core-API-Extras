Выполняется звонок поставщику услуг Интернета...

Будет сделан звонок, чтобы установить уже существующую учетную запись Интернета на этом компьютере.   Щелкните кнопку Далее для набора номера.



Чтобы набрать номер, укажите внизу нужные сведения и нажмите кнопку Далее. Чтобы продолжить работу без подключения к телефонной линии, нажмите кнопку Пропустить.

Выберите страну и регион, в котором вы сейчас находитесь. Для просмотра списка регионов щелкните кнопку со стрелкой или нажмите клавиши ALT + <СТРЕЛКА ВНИЗ>.

Укажите код города (пример: 812 для (812) 530-3496).
