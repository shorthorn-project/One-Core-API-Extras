Активация Windows
Для противодействия компьютерному пиратству корпорация Майкрософт просит вас активировать вашу копию Windows прямо сейчас. Быстрее и проще всего активировать Windows через Интернет. Для дальнейшего использования этой копии операционной системы Windows ее необходимо активировать. Быстрее и проще всего активировать Windows через Интернет.

При активации Windows не требуется сообщать свое имя или другие личные данные.

Активировать Windows сейчас?

  
  
  

Отложив активацию, вы все равно сможете использовать Windows, но будете получать периодические напоминания о необходимости активации. После  дней вы не сможете продолжать работу с Windows до тех пор, пока она не будет активирована.

Корпорация Майкрософт принимает все меры для соблюдения конфиденциальности. Для получения дополнительной информации прочтите Заявление о конфиденциальности активации Windows

Для продолжения нажмите кнопку Далее.
