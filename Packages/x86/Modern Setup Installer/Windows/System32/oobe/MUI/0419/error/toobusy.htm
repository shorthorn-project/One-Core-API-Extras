Телефон занят или не отвечает

Программе установки не удается связаться с этим номером для активации продукта.
Телефон занят или указан неправильный номер.

Если номер правильный, подождите несколько минут и щелкните Набрать номер.

Если номер неправильный, укажите правильный номер и нажмите Набрать номер.
    
    
    
 
 


* Введите альтернативный номер.
* Введите номер для выхода на городскую линию.
* Введите номер для отключения режима принятия входящих звонков.
Чтобы продолжить без активации, щелкните Пропустить.
