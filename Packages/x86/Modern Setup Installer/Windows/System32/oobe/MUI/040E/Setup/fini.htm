Gratulálunk, neki is foghat! A következõket végezte el:
Ha szeretne többet megtudni a Windows XP izgalmas új szolgáltatásairól, nézze meg a programkalauzt. Hasznos információkat talál a súgóban is. Ezeket a funkciókat a Start menüben találja.
A beállítások véglegesítéséhez újra kell indítani a számítógépet.
A számítógép újraindításához kattintson a Befejezés gombra, és máris megkezdheti a Windows használatát.
