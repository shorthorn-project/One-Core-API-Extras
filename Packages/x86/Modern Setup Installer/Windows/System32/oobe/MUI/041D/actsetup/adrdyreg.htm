Onlineaktivering förbereds
En uppringning kommer nu att göras för att aktivera Windows hos Microsoft.

Om du vill fortsätta anger du informationen nedan och klickar på Nästa. Om du vill fortsätta utan att ringa kan du klicka på Påminn mig senare.

Välj det land som du befinner dig i. Klicka på nedåtpilen pilen eller tryck på ALT + NEDPIL om du vill visa listan över regioner.



Skriv in ditt riktnummer.


          Klicka på Nästa om du vill börja ringa upp.
