Det finns ingen kopplingston

Se till att uppringningen görs korrekt genom att kontrollera följande:
  • Telefonkabeln mellan datorn och telefonjacket kanske inte är korrekt ansluten. Läs dokumentationen för att kontrollera att den har anslutits korrekt.



    Obs! Stäng av strömmen om du måste flytta datorn för att nå ett telefonjack. När du startar om datorn återupptar Välkommen till Windows processen där den avslutades.

  • Telefonlinjen kanske används för närvarande. Kontrollera att telefonlinjen är ledig.
