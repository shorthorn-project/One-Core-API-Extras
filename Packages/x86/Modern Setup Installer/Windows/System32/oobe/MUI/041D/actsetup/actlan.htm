Konfigurera nätverksinställningar
Om din anslutning till Internet kräver att du konfigurerar proxyserver kan du ange inställningarna nedan. Kontakta din nätverksadministratör om du är osäker på vilka inställningar du ska använda.

Automatisk konfiguration
Automatisk konfiguration kan åsidosätta de inställningar du gjort själv. Om du anger proxyinställningar själv kan du avmarkera kryssrutan Identifiera inställningar automatiskt för att försäkra dig om att ändringarna tillämpas.
