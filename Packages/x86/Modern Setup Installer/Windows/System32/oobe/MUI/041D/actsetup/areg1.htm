Registrera hos Microsoft?
När du aktiverar Windows online kan du också ange personlig information om dig för att registrera ditt Windows-exemplar.

Registrering är valfri och krävs inte för att aktivera Windows.

När du har registrerat kan Microsoft, med ditt tillstånd, meddela dig om produktuppdateringar, nya produkter och specialerbjudanden.

Vill du registrera samtidigt som du aktiverar Windows?

  
  

Microsoft värnar om din säkerhet. Du kan läsa mer i  sekretesspolicyn för Windows-registrering

Klicka på Nästa om du vill fortsätta.
