Det finns ingen kopplingston

Kopplingston krävs för att anslutningen ska kunna upprättas från din dator. Kontrollera att följande gäller:
  • Telefonlinjen är tillgänglig och kan användas.
  • Telefonlinjen mellan din dator och telefonjacket är korrekt ansluten enligt instruktionerna i datorns dokumentation.




    • Stäng av strömmen om du måste flytta datorn närmare ett telefonjack. När du startar om datorn fortsätter processen där den avslutades.

    Om du har fått det här meddelandet en andra gång, kan du hoppa över detta steg och vi kommer att påminna dig vid ett senare tillfälle.

Klicka på Ring igen om du vill försöka ansluta igen.
Klicka på Hoppa över om du vill hoppa över det här steget.
