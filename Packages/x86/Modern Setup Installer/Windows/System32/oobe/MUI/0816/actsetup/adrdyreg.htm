A preparar a activação online
O Windows irá marcar um número de telefone para activar o Windows junto da Microsoft.

Para avançar, forneça as informações abaixo e clique em Seguinte. Para continuar sem marcar a linha telefónica, clique em Lembrar mais tarde.

Seleccione o 'País' em que se encontra. Clique na seta para baixo ou prima as teclas ALT + "SETA PARA BAIXO" para ver a lista de regiões.



Introduza o indicativo do telefone. (por exemplo, 21 em 212400977).


          Para iniciar a marcação, clique em Seguinte.
