Sinal de linha não detectado

Para garantir que programa de boas vindas ao Windows pode marcar correctamente, verifique o seguinte:
  • O fio telefónico entre o computador e a tomada do telefone poderá não estar ligado correctamente. Consulte a documentação para se certificar de que este está ligado correctamente.



    Nota: Se tiver de mover o computador para chegar a uma tomada de telefone, desligue a alimentação. Quando reiniciar o computador, o programa de boas-vindas ao Windows será retomado no local onde parou.

  • A linha telefónica poderá estar a ser utilizada. Verifique se a linha telefónica está disponível.

Quando estiver preparado para continuar, clique em Marcar novamente.
Para continuar sem configurar o serviço Internet, clique em Ignorar.
