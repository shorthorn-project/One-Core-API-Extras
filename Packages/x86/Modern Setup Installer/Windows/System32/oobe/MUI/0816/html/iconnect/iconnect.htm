Configurar a conta da Internet

Para preencher esta página precisa das seguintes informações do seu fornecedor de serviços Internet (ISP).

     Nome de utilizador: (Exemplo: joaonicolau)
     Palavra-passe de utilizador: (Exemplo: 123xyz)
     Número de telefone do seu ISP:    (Exemplo: 800 | 555-1234)


Se lhe tiverem sido fornecidas as informações que se seguem, introduza-as abaixo. Estas informações são opcionais por isso, se não as tiver, ignore esta secção.

    
Obter o IP automaticamente (DHCP):
   * Introduza um endereço IP válido

     Endereço IP estático:

  Intervalo: 1-223.0-255.0-255.0-255

    
Obter o DNS automaticamente (DHCP):
   * Introduza um endereço DNS válido

     Servidor de nomes de domínio (DNS) prreferido:   

  Intervalo: 1-223.0-255.0-255.0-255
     Servidor de nomes de domínio (DNS) alternativo:   (Opcional)
