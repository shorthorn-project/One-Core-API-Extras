通話に割り込みが入りました


完了前にコンピュータからの電話通話に割り込みが入りました。原因として次のことが考えられます:
  • 長い間、接続を放置していた。
  • 接続中に、別の人が電話を使用した。
  • キャッチホンに呼び出しによる割り込みがあった。このステップが完了するまでキャッチホンは自動的に中止され、完了後にキャッチホンに戻ります。

    • キャッチホンを中止するコード(T)

このメッセージがリダイヤル後にも表示される場合は、このステップを省略して、
後で登録することができます。
