연결이 끊어진 데는 다음과 같은 여러 가지 이유가 있을 수 있습니다.
  • 오랜 시간 동안 연결을 사용하지 않았기 때문에 서버와 연결이 끊어졌을 수 있습니다. 다시 연결하려면 다시 걸기를 클릭하십시오.
  • 연결되어 있는 동안 다른 사용자가 전화를 사용하려고 했습니다. 다시 연결하려면 전화선이 사용 중이 아닌지 확인하고 다시 걸기를 클릭하십시오.
  • 통화 중 대기 서비스를 사용하는 경우에는 걸려온 전화에 의해 연결이 끊어졌을 수 있습니다. 통화 중 대기 서비스 기능을 해제하는 번호를 알고 있다면 아래에 입력한 후, 다시 걸기를 클릭하십시오. 통화 중 대기 서비스 기능을 해제하지 않고 다시 시도하려면 다시 걸기를 클릭하십시오(C).

이 작업을 수행하지 않고 계속 진행하려면 건너뛰기를 클릭하십시오.
