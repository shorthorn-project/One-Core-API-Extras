인터넷 서비스 공급자 선택 인터넷 서비스 공급자에게 문의
귀하의 지역에서 사용 가능한 인터넷 서비스 공급자(ISP)의 목록입니다.

ISP를 눌러 선택하십시오(C). 인터넷 서비스 공급자에서 사용자의 계정 설정을 검색할 수 없습니다. Windows 설정 단계를 마친 후, 인터넷 서비스 공급자에게 문의하여 컴퓨터에 인터넷 계정을 구성하십시오.

 

사용자의 ISP가 목록에 없지만 사용자 이름, 암호, ISP 전화 번호 등을 포함한 인터넷 계정 정보가 있으면 여기를 클릭하십시오.
설정 있음(H)

선택한 ISP의 인터넷 계정 설정을 계속 진행하려면 [다음]을 클릭하십시오.

해당 ISP의 사용을 위해 설치해야 하는 소프트웨어가 있는 경우, [건너뛰기]를 클릭하십시오.
(ISP로부터 CD를 제공받은 경우, [건너뛰기]를 클릭하여야 합니다.)
Windows 설치를 마친 후, 컴퓨터의 인터넷 계정을 설정하십시오.
