Vamos ativar o Windows
Para ajudar a combater a pirataria de software, ative a sua cópia do Windows agora. A ativação pela Internet é rápida e fácil. Esta cópia do Windows precisa ser ativada na Microsoft antes que você possa continuar a usá-la. A ativação pela Internet é rápida e fácil.
