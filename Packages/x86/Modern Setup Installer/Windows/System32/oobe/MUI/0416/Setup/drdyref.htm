Você precisa de ajuda para encontrar um provedor de serviços de Internet?


Se você precisar de ajuda para encontrar informações sobre o seu provedor de serviços de Internet para poder copiar sua conta de Internet para o computador, é possível fazer uma chamada telefônica ao serviço de referência da Microsoft para procurar o provedor.

Você precisa de ajuda para configurar sua conta na Internet?
  
  
Para continuar, forneça as informações abaixo e clique em Avançar. Para continuar sem discar na sua linha telefônica, clique em Ignorar.

Selecione o país ou região onde você está agora. Clique na seta para baixo ou pressione as teclas ALT + "seta para baixo" para exibir a lista de regiões.



Digite o código de área do seu telefone (por exemplo, 99 em 99-222-3333).
