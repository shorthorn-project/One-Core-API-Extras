Sinal de discagem não detectado

Para certificar-se de que o Windows discará corretamente, verifique:
  • O fio de telefone que liga o computador à tomada telefônica pode não estar conectado corretamente. Consulte a documentação e verifique se sua conexão está correta.



    Observação: se você precisar mover o computador para conectá-lo a uma tomada telefônica, desligue-o primeiro. Quando o computador for reiniciado, o Windows continuará do ponto em que você parou.

  • A linha telefônica pode estar sendo usada. Verifique se ela está disponível.

Quando você estiver pronto para continuar, clique em Rediscar.
Para continuar sem configurar o serviço de Internet, clique em Ignorar.
