Deseja se registrar na Microsoft?
Enquanto estiver ativando o Windows on-line, você terá a chance de fornecer seus dados pessoais a fim de registrar sua cópia do Windows.

O registro é opcional e não é necessário para ativar o Windows.

Depois que você estiver registrado, a Microsoft poderá entrar em contato com você, se tiver o seu consentimento, com informações sobre atualizações, novos produtos, eventos e ofertas especiais.

Deseja fazer o registro do Windows junto com a ativação?

  
  

A Microsoft tem um compromisso com a sua privacidade. Para obter mais informações,  leia a declaração de privacidade de registro do Windows
