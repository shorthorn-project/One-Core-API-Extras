Internetkonto einrichten

Sie müssen die folgenden Informationen von Ihrem Internetdienstanbieter beziehen, um diesen Schritt durchzuführen.

     Benutzername: (Beispiel: jensmander)
     Kennwort: (Beispiel: 123xyz)
     Rufnummer für den Internetdienstanbieter:    (Beispiel: 0800 | 892267)


Falls Sie über die folgenden Angaben verfügen, geben Sie diese ein. Falls Sie über diese Angaben nicht verfügen, können Sie diesen Abschnitt ignorieren.

    
IP-Adresse automatisch beziehen (DHCP):
   * Geben Sie eine gültige IP-Adresse ein

     Statische IP-Adresse:

  Bereich: 1-223.0-255.0-255.0-255

    
DNS automatisch beziehen (DHCP):
   * Geben Sie eine gültige DNS-Adresse ein

     Bevorzugter DNS-Server:   

  Bereich: 1-223.0-255.0-255.0-255
     Alternativer DNS-Server:   (optional)
