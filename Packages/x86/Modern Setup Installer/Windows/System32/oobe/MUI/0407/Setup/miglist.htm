Internetdienstanbieter auswählen An den Internetdienstanbieter wenden
Unten wird ein Teil der Internetdienstanbieter in Ihrer Nähe angezeigt. Es werden nur nationale Internetdienstanbieter in dieser Liste angezeigt.

Wählen und klicken Sie auf einen Internetdienstanbieter. Die Kontoeinstellungen können nicht von Ihrem Internetdienstanbieter abgefragt werden. Wenden Sie sich an Ihren Internetdienstanbieter, um Ihr Internetkonto auf diesem Computer zu konfigurieren, nachdem dieser Einführungsvorgang abgeschlossen wurde.

 

Klicken Sie hier, falls Ihr Internetdienstanbieter nicht in der Liste aufgeführt ist, Sie aber über die Internetkontoinformationen, einschließlich Benutzernamen, Kennwort und die Rufnummer des Internetdienstanbieters, verfügen:
Informationen werden bereit gehalten

Klicken Sie auf "Weiter", um ein Internetkonto mit dem gewählten Internetdienstanbieter zu konfigurieren.

Klicken Sie auf "Überspringen", falls für die Internetkontoeinrichtung zusätzliche Software installiert werden muss.
(Falls dies der Fall ist, haben Sie eine CD von Ihrem Internetdienstanbieter erhalten.)
Sie sollten erst Windows einrichten, und dann den Computer für den Internetzugang konfigurieren.
