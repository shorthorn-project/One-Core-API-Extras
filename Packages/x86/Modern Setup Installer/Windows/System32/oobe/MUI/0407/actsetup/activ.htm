Möchten Sie Windows jetzt aktivieren?

  
  
  

Falls Sie Windows jetzt nicht aktivieren, können Sie es weiterhin verwenden, aber Sie werden in regelmäßigen Abständen erinnert, Windows zu aktivieren.
In  Tag(en) muss Windows aktiviert werden, sonst kann es nicht mehr verwendet werden. Falls Sie Windows telefonisch über den Kundendienst aktivieren möchten, sollten Sie die Aktivierung jetzt nicht ausführen und die Option "Nein, in ein paar Tagen erinnern" wählen.
