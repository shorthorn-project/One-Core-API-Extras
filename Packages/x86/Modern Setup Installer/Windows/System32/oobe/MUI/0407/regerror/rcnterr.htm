Die Verbindung wurde unterbrochen
Es kann mehrere Gründe geben, warum die Verbindung unterbrochen wurde:
  • Der Server hat die Verbindung aufgrund längerer Zeit ohne Aktivitäten möglicherweise gelöscht. Klicken Sie auf Wahlwiederholung, um erneut eine Verbindung herzustellen.
  • Möglicherweise hat jemand versucht, Ihre Telefonleitung zu verwenden, während Sie verbunden waren. Überprüfen Sie, ob die Telefonleitung verfügbar ist, und klicken Sie auf Wahlwiederholung, um erneut eine Verbindung herzustellen.
  • Bei Anklopffunktion wurde die Verbindung möglicherweise von einem ankommenden Anruf unterbrochen. Wenn Sie die Nummer kennen, um die Anklopffunktion auszuschalten, geben Sie sie hier ein und klicken Sie auf Wahlwiederholung. Wenn Sie die Nummer nicht kennen, klicken Sie auf Wahlwiederholung, um erneut eine Verbindung herzustellen.

Klicken Sie auf Überspringen, um den Vorgang fortzusetzen, ohne diese Aufgabe durchzuführen.
