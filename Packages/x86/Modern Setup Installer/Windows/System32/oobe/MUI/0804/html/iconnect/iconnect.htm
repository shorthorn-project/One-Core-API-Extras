设置 Internet 帐户

您需要从 Internet 服务提供商(ISP)那里得到以下信息来填写此页。

     您的用户名(U): (例如: johndoe)
     您的用户密码(W): (例如: 123xyz)
     您的 ISP 电话号码(P):    (例如: 800 | 555-1234)


如果获得下列信息，请输入在下面。该信息为可选的，如果您没有，则不要管这部分。

    
自动获取 IP(DHCP)(I):
   * 请输入有效 IP 地址

     静态 Internet 协议(IP)地址(C):

  范围: 1-223.0-255.0-255.0-255

    
自动获取 DNS(DHCP)(D):
   * 请输入有效 DNS 地址

     首选域名服务器(DNS)(R):   

  范围: 1-223.0-255.0-255.0-255
     备用域名服务器(DNS)(A):   (可选项)
