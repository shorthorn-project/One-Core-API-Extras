选择您的 Internet 服务提供商 与您的 Internet 服务提供商联系
以下是您所在地区的一部分 Internet 服务提供商(ISP)列表。不提供全国服务的 ISP 不在此表中。选择一个 ISP (单击一个)(C)。 Windows 无法从您的 Internet 服务提供商中检索您的帐户设置，Windows 完成后，请与您的 Internet 服务提供商联系，咨询如何在该计算机上配置您的 Internet 帐户。

 

如果您的 ISP 不在表上，但您手边已有您 Internet 帐户的信息，包括您的用户名、密码、和 ISP 的电话号码，请单击此处:
设置(H)

要继续用您所选 ISP 设置 Internet 帐户，单击“下一步”。

如果您想要用的 ISP 要求您安装它自己的软件，单击“跳过”。
(您会知道如此，如果您已有那 ISP 的 CD。)
安装好 Windows 后，再设置您的 PC 访问 Internet。
