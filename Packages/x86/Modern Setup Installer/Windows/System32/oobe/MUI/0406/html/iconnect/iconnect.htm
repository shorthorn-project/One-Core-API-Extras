Konfigurer din Internetkonto

Du skal bruge følgende oplysninger fra din Internetudbyder for at kunne udfylde denne side.

     Dit brugernavn: (Eksempel: jensjensen)
     Din brugeradgangskode: (Eksempel: 123xyz)
     Din Internetudbyders telefonnummer:    (Eksempel: 31 | 32 33 34)


Hvis du har fået følgende oplysninger, kan du angive dem nedenfor. Det er valgfrit, om du vil angive oplysninger, så hvis du ikke har dem, kan du springe dette afsnit over.

    
Hent IP automatisk (DHCP):
   * Angiv en gyldig IP-adresse

     Statisk IP-adresse (Internetprotokol):

  Område: 1-223.0-255.0-255.0-255

    
Hent DNS automatisk (DHCP):
   * Angiv en gyldig DNS-adresse

     Foretrukken DNS (Domain Name Server):   

  Område: 1-223.0-255.0-255.0-255
     Alternativ DNS (Domain Name Server):   (valgfri)
