Windows activeren
Activeer nu uw exemplaar van Windows om zo de verspreiding van illegale kopieën tegen te gaan. Activeren via het Internet is snel en gemakkelijk. Dit exemplaar van Windows moet, voordat u het verder kan gebruiken, bij Microsoft worden geactiveerd. Activeren via het Internet is snel en gemakkkelijk.

U hoeft uw naam of andere persoonlijke gegevens niet op te geven als u Windows activeert.

Wilt u Windows nu activeren?

  
  
  

Als u wacht met activeren, kunt u Windows wel blijven gebruiken maar zult u periodieke herinneringen ontvangen.
Na  dag(en) dient u Windows te activeren voordat u het besturingssysteem verder kunt gebruiken.

Microsoft hecht zeer aan de bescherming van uw privacy. Zie voor meer informatie  de privacyverklaring voor het activeren van Windows.
