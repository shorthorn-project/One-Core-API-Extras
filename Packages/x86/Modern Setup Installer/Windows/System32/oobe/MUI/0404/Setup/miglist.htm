選擇您的網際網路服務提供者 請連絡您的網際網路服務提供者
下面是您所在區域的網際網路服務提供者 (ISP) 的部分名單。不是全球性的 ISP 將不會顯示在這份名單中。

請選擇一個 ISP (在名單項目上按一下)(C)。 Windows 歡迎畫面無法從您的網際網路服務提供者擷取您的帳戶設定。在 Windows 歡迎畫面完成之後，請連絡您的網際網路服務提供者，請他們協助您在這台電腦上設定您的網際網路帳戶。

 

如果您的 ISP 不在這份名單中，但您手邊有您的網際網路帳戶資料，包括使用者名稱、密碼及 ISP 的撥接電話。請按這裡:
已有設定(H)

如果要繼續用您所選擇的 ISP 來設定網際網路帳戶，請按 [下一步]。

如果這個 ISP 要求您安裝它的軟體，請按 [跳過]。
(如果您收到 ISP 寄來的安裝光碟，就表示您必須安裝它的軟體。)
然後當您完成 Windows 設定時，再設定電腦的網際網路存取。
